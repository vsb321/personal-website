Description

Soliman is a beautiful easy to use Portfolio template built on bootstrap The template is responsive, so you can view it also in the mobile/tablets devices and it looks very nice in those devices. 

The template have :

  1. Responsive layout
  2. W3C valid HTML files
  3. Html 5 & CSS3
  4. Unique design
  5. and much more...

Loading Screen

Turn Off Loading Screen

Open the html file using any text editor. We recommend Sublime Text. Now look for the following code and remove them.

<!--Start Preloader-->

<div class="page-loader startLoad">

  <div class="loader startLoad"></div>

</div>

<!--End Preloader-->

Save the file and refresh your browser.
